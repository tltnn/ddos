Documentación de la API

GET /api/ytmp3?query={search_query}

Obtiene información y enlace de descarga MP3 para un video de YouTube basado en una búsqueda.

Parámetros:

Parámetro Tipo Requerido Descripción
query string Sí Término de búsqueda o URL de YouTube
quality string No Calidad de audio (low, medium, high). Default: high

La API devuelve un objeto JSON con la siguiente estructura:

{ "success": boolean, "video": { "title": string, "duration": string, "views": string, "author": string, "published": string, "thumbnail": string }, "download": { "type": "audio", "format": "mp3", "url": string, "filename": string, "filesize": number } }

Ejemplo de respuesta exitosa:

{ "success": true, "video": { "title": "Enemy - Imagine Dragons", "duration": "2:58", "views": "1.2B views", "author": "ImagineDragonsVEVO", "published": "2 years ago", "thumbnail": "https://i.ytimg.com/vi/D9G1VOjN_84/maxresdefault.jpg" }, "download": { "type": "audio", "format": "mp3", "url": "https://api.example.com/download/abc123", "filename": "Enemy - Imagine Dragons.mp3", "filesize": 4567890 } }

Ejemplo de respuesta con error:

{ "success": false, "error": "No se encontraron resultados para la búsqueda", "code": 404 }
GET /api/ytmp4?url={youtube_url}

Obtiene información y enlaces de descarga para un video de YouTube en formato MP4.

Parámetros:

Parámetro Tipo Requerido Descripción
url string Sí URL completa del video de YouTube
quality string No Filtrar por calidad (144p, 240p, 360p, 480p, 720p, 1080p, etc)

La API devuelve un objeto JSON con la siguiente estructura:

{ "success": boolean, "video": { "title": string, "duration": number, "views": number, "author": string, "thumbnail": string }, "downloads": [ { "quality": string, "format": string, "url": string, "filesize": number, "fps": number } ] }

Ejemplo de respuesta exitosa:

{ "success": true, "video": { "title": "Enemy - Imagine Dragons", "duration": 178, "views": 1200000000, "author": "ImagineDragonsVEVO", "thumbnail": "https://i.ytimg.com/vi/D9G1VOjN_84/maxresdefault.jpg" }, "downloads": [ { "quality": "720p", "format": "mp4", "url": "https://api.example.com/download/def456", "filesize": 12345678, "fps": 30 }, { "quality": "1080p", "format": "mp4", "url": "https://api.example.com/download/ghi789", "filesize": 23456789, "fps": 60 } ] }

Ejemplo de respuesta con error:

{ "success": false, "error": "URL de YouTube no válida", "code": 400 }
