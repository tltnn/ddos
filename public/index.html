Documentación de la API

Endpoints disponibles para interactuar con el servicio de YouTube. Todos los endpoints requieren autenticación mediante API key.

GET /api/ytmp3?query={search_query}&quality={quality}&format={format}

Obtiene información y enlace de descarga MP3 para un video de YouTube basado en una búsqueda o URL.

Parámetros:

Parámetro Tipo Requerido Descripción
query string Sí Término de búsqueda o URL completa de YouTube
quality string No Calidad de audio (high, medium, low). Default: high
format string No Formato de salida (mp3, m4a, wav). Default: mp3

Ejemplo de solicitud:

GET /api/ytmp3?query=Enemy%20Imagine%20Dragons&quality=high&format=mp3 Host: api.example.com Authorization: Bearer your_api_key

Ejemplo de respuesta:

{ "success": true, "video": { "title": "Enemy - Imagine Dragons", "duration": "2:58", "views": "1.2B views", "author": "ImagineDragonsVEVO", "published": "2 years ago", "thumbnail": "https://i.ytimg.com/vi/D9G1VOjN_84/maxresdefault.jpg" }, "download": { "type": "audio", "format": "mp3", "url": "https://api.example.com/download/abc123", "filename": "Enemy - Imagine Dragons.mp3", "filesize": 4567890 } }
GET /api/ytmp4?url={youtube_url}&quality={quality}

Obtiene información y enlaces de descarga para un video de YouTube en formato MP4 con diferentes calidades.

Parámetros:

Parámetro Tipo Requerido Descripción
url string Sí URL completa del video de YouTube
quality string No Filtrar por calidad específica (144p, 240p, 360p, 480p, 720p, 1080p, etc)

Ejemplo de respuesta:

{ "success": true, "video": { "title": "Enemy - Imagine Dragons", "duration": 178, "views": 1200000000, "author": "ImagineDragonsVEVO", "thumbnail": "https://i.ytimg.com/vi/D9G1VOjN_84/maxresdefault.jpg" }, "downloads": [ { "quality": "720p", "format": "mp4", "url": "https://api.example.com/download/def456", "filesize": 12345678, "fps": 30 }, { "quality": "1080p", "format": "mp4", "url": "https://api.example.com/download/ghi789", "filesize": 23456789, "fps": 60 } ] }
